Under the Hood

FocusGuard uses a 3-stage agentic AI pipeline powered by Groq's LPU for sub-200ms response times. When distraction is detected, the image is analyzed by Llama 4 Scout, a personalized message is crafted by Llama 4 Maverick, and safety is ensured by Llama Guard 4.

Tech Stack

Built for judges who appreciate good architecture

Frontend

  • Vanilla JavaScript (no framework bloat)
  • MediaPipe FaceMesh (iris tracking)
  • Web Speech API (text-to-speech)
  • WebSocket (real-time comms)

Backend

  • Python + FastAPI
  • Groq SDK (LPU inference)
  • Opik (LLM observability)
  • Uvicorn ASGI server

AI Models

  • Llama 4 Scout 17B (vision)
  • Llama 4 Maverick 17B (reasoning)
  • Llama Guard 4 12B (safety)
  • All via Groq Cloud
